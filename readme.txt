STRUNK AND WHITE THEME

Thanks for using the Strunk & White theme. This WordPress theme was designed for writers and readers, but you can use it for any website.

The theme's main informational page is at:
http://www.zenwebthemes.com/shop-themes/strunk-white/

------------------------------
FREE AND PREMIUM VERSIONS
------------------------------

This theme comes in two versions: Free and Premium.

Free Features

- Large homepage photo 
- Responsive design
- A social media widget
- A widgetized homepage that makes it easy to set up your site
- Call-to-Action button and headline widget

Premium Features

The Premium version has all of the free features, plus:

- Choose your own accent color
- Google Fonts
- Typography choices such as colors and sizes
- Full width, left sidebar, and right sidebar Page layouts
- Different sidebars for Posts vs Pages
- Open content widget areas on the homepage
- A widget area for the end of all blog posts
- Tech support
